# minimal-masonry

Minimal implementation of the [masonry layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout/Masonry_Layout).

Note: There is a W3C working draft for the masonry layout ([CSS Grid Lanes](https://www.w3.org/TR/css-grid-3/#grid-lanes-layout)) as of late 2025. However, it is [only implemented in Safari](https://caniuse.com/css-grid-lanes) as of March 2026. For more context: [When will CSS Grid Lanes arrive? How long until we can use it?](https://webkit.org/blog/17758/when-will-css-grid-lanes-arrive-how-long-until-we-can-use-it/)
